Bomb!


This is a game that you have to find all bombs without one mistake.

There are 100 blocks for you to choose.


If you click a place that there is no bomb aroud, it will open all the box near it. Or it will tell you how many bombs near it.

If you kow where the bomb is, click the bomb checkbox below and click the box. Click again to undo.


If you click the bomb, you lose.
It will show you all the position of the bombs.

If you finish all 100 block and find out all the content in the blocks, you win.
