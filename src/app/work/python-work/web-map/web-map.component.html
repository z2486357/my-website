Web Map


This is a html file that generated and coded by python.
It is a world map and have some functions.

You can put some different pin on it. Also, change attribute due to some criteria. (Change color due to height in this case.)



You can color the map according to different criteria. (Population in this case.)



And you can put all these into the options so that you can turn on and off at any time.



Also you can pop up some information when you click the pin. (Height of the volcano in this case.)
