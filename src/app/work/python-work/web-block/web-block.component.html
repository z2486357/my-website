Website Blocker


This is an application that you can block out some website in the determine time (When you are working / When you are not on the seat / When someone borrow your computer / ...... ).

1. Declare the website you want to block.

2. Determine the time to start and end.

3. Let it start automatically when turning on computer by pressing start button.
