Interactive English Dictionary


This is an English dictionary, you can type the word you would like to know and press Enter for the explanation.


For example, the explanation of state.
You can use the scroll bar if the explanation is too much.


If the world can not be found but it is close to another word. It will ask "Did you mean xxx ?"


If you press Yes, It will show it.

If you press No, It will show "Not Found".
