Board Reader


This are two main part about this application.
1. Get the position of the 100 points on the board
2. Use the position to get the data you want in txt file and output to excel file.

Part 2. Get the data

The data recorded in the txt file include the position x and position y of the picture and the x,y,z value of the point in the real world.
After getting the position data from part 1, the application will go into the txt file to find the x,y,z value. If there is no value of the point, it will find the nearest point which include a value.

Part 3. Output the result

After getting the value, th application will output an excel file that include the origin 100 points, the 100 points where the value is got and the x,y,z value.
