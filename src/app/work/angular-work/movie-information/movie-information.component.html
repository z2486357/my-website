Movie Information

You can visit this web application on http://movie-information-angular.s3-website.us-east-2.amazonaws.com/

Also, you can find the code on https://github.com/z2486357/MoviePost

This project was coded in Angular. The API and data used in this application was from the website "The Movie Database" https://www.themoviedb.org/

When you enter the website, you will see the popular movies of the time. Including its poster, overview and some information.

If you click the post, you can enter to the detail of the movie.

At the bottom, there are some movies which are similar to this movie. Click it and you can enter the detail of the chosen movie.

You can search the target movie by using the search bar on the top left corner. Press the search button without any input can bring you back to the popular movies.

You can use the "<" and ">" to change the page.

Or input the page and press Go button to go to the page you want.
